# Plan del proyecto

Objetivo
- Crear un portfolio personal sencillo que muestre trabajos de dirección creativa y diseño urbano.

Estructura
- `index.html`: HTML principal con secciones Inicio, Sobre mí, Proyectos y Contacto.
- `assets/css/styles.css`: estilos personalizados.
- `assets/js/main.js`: interacciones y animaciones (GSAP).

2. Añadir más proyectos y ejemplos visuales.
3. Revisar responsive y accesibilidad.

Notas
