# Plan 1 — Desarrollo de la plantilla de portfolio

## Contexto

Primer uso del flujo de trabajo con IA propuesto en la asignatura **Web Atelier (UDIT)**. Este plan documenta de forma sencilla cómo se utilizó un asistente de IA para **pensar la estructura y el enfoque general** del portfolio antes de escribir el código.

---

## Fase 1 — Planificación (uso de IA)

**Prompt utilizado (resumen):**

> Ayúdame a definir la estructura de una plantilla de portfolio profesional en HTML, CSS y JavaScript vanilla, con un estilo creativo/brutalista, animaciones básicas y cumpliendo los requisitos del brief del curso.

**Resultado de la planificación:**

* Definir una estructura HTML semántica clara (header, main, sections, footer).
* Usar una arquitectura CSS simple con variables, layout y componentes.
* Priorizar un diseño responsive mobile-first.
* Incluir animaciones CSS y JavaScript ligeras (hero y proyectos).
* Mantener el proyecto en vanilla, sin frameworks ni build tools.

---

## Fase 2 — Implementación

A partir del plan anterior se implementó:

* Estructura base del proyecto y despliegue en GitHub Pages.
* Layout principal con secciones: Hero, About, Projects y Footer.
* Sistema tipográfico y paleta de color coherentes.
* Animaciones sencillas para el hero y la interacción con proyectos.
* Ajustes manuales posteriores para mejorar legibilidad, diseño y coherencia visual.

No todo lo sugerido por la IA se implementó literalmente; el plan se usó como **guía inicial**, y muchas decisiones se tomaron manualmente durante el desarrollo.

---

## Notas finales

Este plan refleja un primer acercamiento al uso de IA como herramienta de apoyo en la **fase de planificación**, no como generador automático de código. El objetivo principal fue mantener el control del diseño y del código, utilizando la IA solo como ayuda para estructurar el trabajo.
